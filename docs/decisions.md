# Re:Memo 設計判断ログ

## ADR-001: AI推論にGemini 2.5 Flashを採用
- **決定日**: 2026-04-02
- **理由**: コスト効率が高く、JSON出力が安定。要点抽出・採点の両方に対応可能。
- **代替案**: GPT-4o-mini（コスト高）、Claude Haiku（JSON mode未対応時期あり）

## ADR-002: SRSアルゴリズムにSM-2を採用
- **決定日**: 2026-04-02
- **理由**: 実績あり、乙4マスターで検証済み。FSRS等の新手法は将来検討。
- **パラメータ**: 初期EF=2.5, 最小EF=1.3, 初回1日, 2回目6日

## ADR-003: 声・テキスト回答のポイント消費を同じ1ptに設定
- **決定日**: 2026-04-02
- **理由**: 声での回答を促進するため、コスト差（Deepgram API分）はサービス側で吸収。
- **XP差**: 声=20XP, テキスト=10XP で声を間接的にインセンティブ。

## ADR-004: 録音機能はv2以降に延期
- **決定日**: 2026-04-02
- **理由**: MVP範囲を絞るため。メモ・YouTube・PDFの3ソースで先にリリース。

## ADR-005: 音声入力にDeepgram Nova-3を採用
- **決定日**: 2026-04-02
- **理由**: 日本語認識精度が高く、リアルタイムSTT対応。Whisperより低レイテンシ。
